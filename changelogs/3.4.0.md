## [3.4.0](https://github.com/Kevin-Lee/sbt-devoops/issues?q=is%3Aissue%20is%3Aclosed%20-label%3Adeclined%20milestone%3Am42) - 2026-02-22

### New Features

* `devOopsGitHubRelease` should update the changelog if the target release already exists (#502)

  When the target release already exists,
  
  **Before**
  ```
  [error] >> Failed:
  [error] Unprocessable Entity:
  [error] responseBody: {
  [error]   "message" : "Validation Failed",
  [error]   "errors" : [
  [error]     {
  [error]       "resource" : "Release",
  [error]       "code" : "already_exists",
  [error]       "field" : "tag_name"
  [error]     }
  [error]   ],
  [error]   "documentation_url" : "https://docs.github.com/rest/releases/releases#create-a-release"
  [error] }
  ```
  
  **After**
  ```
  >> task success>
  >> Try to create a GitHub release with tag: v0.1.2
  
  >> task success>
  >> Release with tag already exists. Try to find the existing release by tag: v0.1.2
  
  >> task success>
  >> Found existing release id: 123456789. Try to update release body.
  
  >> task success>
  >> Update release API succeeded for release id: 123456789 (tag: v0.1.2)
  
  >> task success>
  >> GitHub release updated: v0.1.2
  ```
